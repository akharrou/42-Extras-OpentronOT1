# Opentrons OT-1: Autonomous Biological Protocol

A custom liquid handling and sample extraction workflow for the legacy Opentrons OT-1 robotic platform. Automates the precise transfer of living mycelium from standard petri dishes into 96-well trays, executing the extraction sequence in an Archimedean spiral motion.

Includes: algorithmic path planning, serial hardware control, sub-millimeter deck calibration, and custom 3D-modeled labware fixtures.

### Automated Run Demonstration

The system operates autonomously. In the demonstration below, the OT-1 is driven directly by a Raspberry Pi executing the Python protocol sequence.
